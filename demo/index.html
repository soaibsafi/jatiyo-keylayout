আপনার প্ল্যাটফর্ম বেছে নিন এবং নির্দেশনা অনুসরণ করুন
Homebrew

Terminal-এ নিচের কমান্ডগুলো ক্রমান্বয়ে চালান:

brew tap soaibsafi/jatiyo
brew install --cask jatiyo
অথবা
ইনস্টলার প্যাকেজ .pkg

রিলিজ পেজ থেকে Jatiyo-Installer.pkg ডাউনলোড করুন এবং ডাবল-ক্লিক করে ইনস্টল করুন। ইনস্টলের পরে System Settings → Keyboard → Input Sources-এ কীবোর্ড যোগ করুন।

Releases থেকে ডাউনলোড

প্রথমে keyman.com থেকে Keyman for Windows ইনস্টল করুন। তারপর রিলিজ পেজ থেকে jatiyo.kmp ফাইলটি ডাউনলোড করুন — Keyman স্বয়ংক্রিয়ভাবে ইনস্টল করবে।

jatiyo.kmp ডাউনলোড

প্রথমে Keyman for Linux ইনস্টল করুন: 

sudo add-apt-repository ppa:keymanapp/keyman
sudo apt update && sudo apt install keyman

তারপর jatiyo.kmp ডাউনলোড করে Keyman-এ ইনস্টল করুন:

km-package-install ./jatiyo.kmp
Android ও iOS

Play Store বা App Store থেকে Keyman অ্যাপ ডাউনলোড করুন। তারপর রিলিজ পেজ থেকে jatiyo.kmp ডাউনলোড করুন — Keyman অ্যাপ স্বয়ংক্রিয়ভাবে ইনস্টল করবে।

Google Play (Keyman) App Store (Keyman) jatiyo.kmp ডাউনলোড
Keyman Web SDK

আপনার HTML পেজে keymanweb.js ও jatiyo.js যোগ করুন:

<script src="https://s.keyman.com/kmw/engine/17.0.326/keymanweb.js"></script>

তারপর JavaScript-এ কীবোর্ড সক্রিয় করুন:

keyman.init({ attachType: 'auto' });
keyman.addKeyboards({
  id: 'jatiyo',
  name: 'Bangla Jatiyo',
  language: { id: 'bn', name: 'Bengali' },
  filename: './jatiyo.js'
});
সোর্স থেকে বিল্ড

প্রজেক্ট ক্লোন করুন:

git clone https://github.com/soaibsafi/jatiyo-keylayout.git

Keyman প্যাকেজ বিল্ড করুন:

cd keyman/source && kmc build jatiyo.kps

macOS নেটিভ ইনস্টলার বিল্ড করুন:

cd macos-native && ./build-installer.sh
