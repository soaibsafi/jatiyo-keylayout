Welcome to the Jatiyo Installer

This installer adds the Jatiyo Bengali (বাংলা) keyboard layout to your Mac.

What this installer does

  • Copies jatiyo.keylayout to /Library/Keyboard Layouts/
  • Copies jatiyo.icns to /Library/Keyboard Layouts/

After installation

  1. Open System Settings → Keyboard → Input Sources → Edit
  2. Click the + button
  3. Choose Other and select বাংলা-জাতীয়

You may be asked for your administrator password to install into a protected system folder.
